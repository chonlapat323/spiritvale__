# Tier List (Theorycraft — no community parses yet)

All rankings below are speculative, based on kit/skill-name analysis and dev patch notes, not measured damage/parses. Revisit once real numbers or community meta appear.

## v1 — Day-1, pre-patch-analysis
- **S**: Weaver — unlocks far later than others, huge skill pool, any-class grimoire flexibility → likely strongest/most versatile endgame class
- **A**: Paladin (tank+heal+buff hybrid), Wizard (strong AOE elemental burst)
- **B**: Gunslinger (most skills, good control), Shinobi (single-target burst/mobility), Priest (essential pure healer, weak solo)
- **C**: Berserker (glass cannon, risky scaling), Necromancer (minion AI/scaling risk)

## v2 — post balance-patch (see [patch-notes.md](patch-notes.md))
- **S**: Weaver, Gunslinger (moved up — deep rework, burst+control kit)
- **A**: Wizard, Paladin (Sacrifice rework restores real offense — see [classes/paladin.md](classes/paladin.md); net unchanged from v1 after full analysis)
- **B**: Shinobi, Priest
- **C**: Berserker, Necromancer (Ghost element shift is a wildcard — could move up if Ghost is under-resisted)

## Ranked pick list (for choosing a class fresh)
1. Weaver (S) — best long-term/endgame flexibility, but very high level gate (Lv131-135), not beginner-friendly
2. Gunslinger (S) — deep rework, rewards planning/combos, ranged playstyle
3. Wizard (A) — strong AOE elemental farm/burst, simple and effective, but squishy
4. Paladin (A) — tank with real DEF-ignoring damage via Sacrifice, good for players who want to tank but not be damage-dead weight
5. Shinobi (B) — high single-target burst/mobility, PvP/boss-favored, weaker vs groups
6. Priest (B) — essential party healer, weak solo/leveling
7. Necromancer (C, wildcard) — minion+lifesteal attrition, Ghost element rework could shift this up or down
8. Berserker (C) — high-risk glass cannon, damage scales with "the fight getting bloodier"
