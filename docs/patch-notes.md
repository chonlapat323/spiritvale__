# Balance Patch (user-provided, direct from dev — reported same day as launch, version/date TBD)

Trust this over the wiki when they conflict — see [README.md](../README.md) source priority note.

## Progression
- EXP requirements for higher levels up ~50%
- Essence drop rates reduced to 0.1% (all)
- Essence crafting now costs 25k-100k coins
- Equipment drop rates slightly reduced
- Crafting material costs ~2x
- Boss Crystal drop rate 200% → 150%
- Weekend Bonus EXP removed
- Party EXP share reduced beyond 30-level gap, zeroed at 50-level gap (Lv130+ all counts as Lv130)

→ Overall reading: broad progression nerf/slowdown (slower leveling, slower gearing, higher gold sink). Grouping with wide level gaps now much less efficient.

## Ghost Element (game-mechanics)
- New Ghost element added, element table reworked
- Many monsters converted to Ghost element
- Summoner skills converted to Ghost element

→ This directly affects Necromancer (promotes from Summoner) — if Summoner's damage skills are now Ghost-typed, Necromancer's offense likely also shifts to Ghost. Need to check elemental resist chart once available; could be a buff or nerf depending on how common Ghost resist is on mobs/gear.

## Cards and Gear (global)
- Added/moved many cards
- Removed Skill Damage cards
- Removed Weapon Element cards
- Equipment Potential values reduced
- Neutral resist excluded from all resist mods
- Siphon scaling halved
- Autocast changes: removed autocast headgear cards, removed Bone Pick autocast; new autocast rates — Tomahawk 15%, Scalpel 15%, Frost Fang 15% (Frost Fang disables status from ice shard)

→ Net effect: several damage-scaling levers (Skill Damage cards, Weapon Element cards, Potential, Siphon) all pulled back simultaneously = broad damage-inflation curb across the board, not class-specific.

## Paladin — mixed, leans nerf on offense/utility, buff on block/shields
- Sacrifice: reworked to single-target attack, 5-10% target HP as damage, ignores defense, triggers hit effects (was presumably a self-sacrifice/support skill before — now an offensive tool)
- Shield skills: damage +30% (buff)
- Consecration/Heal: lost regeneration buff (nerf)
- Holy Shield: lost all-resistance buff (nerf)
- Oathbreaker: lost +1 matk per str, gained 10% block instead (shift from hybrid caster-tank to pure block-tank)
- Judgement Blade: damage -25%, no longer triggers hit effects (nerf)
- Grand Cross: no longer triggers hit effects (nerf)

→ See [classes/paladin.md](classes/paladin.md) for full deep-dive and reassessment.

## Gunslinger — complete rework, direction unclear without numbers but structurally deeper
- All guns unified to 5% dex scaling (simplifies itemization)
- Auto-swap to correct loadout per skill (quality-of-life buff)
- New Heavy Loadout unlocks Launcher/Gatling as a 3rd weapon set
- Shotgun range increased
- Notable skill changes: Lockdown reworked from a hard-CC lock into a stacking self-buff (movement-lock removed, now rewards standing still: +30% multi, +15% atk, +20% aspd x10 stacks) — shifts Gatling from control tool to a channel-buff-then-burst tool
- Launcher grenades reworked with clearer buff windows (Launcher Boost from Explosive Grenade: 175 aspd + 100% autocast Explosive Grenade for 3s; Launcher Poison from Poison Grenade: +50% status dmg)
- Utility skills (Jump Shot, Aerial Shot) now usable always (removes prior restrictions — buff to uptime/mobility)

→ **Reassessment: Gunslinger moves up from B to A/S.** A "complete rework" this deep, combined with QoL buffs (auto-swap, always-usable utility) and clearer burst windows (Launcher Boost, Lockdown stacks) suggests Gunslinger was pushed toward being a top-end burst/control hybrid. Caution: without hard numbers this is still theorycraft, but the sheer scope of investment devs put into this class is itself a signal.
